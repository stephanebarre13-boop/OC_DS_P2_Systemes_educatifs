# Projet 2 — Analysez des données de systèmes éducatifs

**Formation** : Data Scientist — OpenClassrooms  

## Contexte

Analyse des données de la Banque Mondiale (EdStats) pour identifier 
les pays à fort potentiel d'expansion pour une startup EdTech.

## Compétences mises en œuvre

- Exploration et nettoyage de données avec Python/Pandas
- Analyse statistique descriptive
- Visualisations avec Matplotlib/Seaborn
- Identification de patterns et recommandations
